Artificial Intelligence - A Star Search through two Application

Compile before launching :

javac *.java

The tests folder contains txt files containing the worlds/grids on which to run the program. Files containing '+' contain the expected results.

For DemoA (nPuzzle) here is the format :
java demoA tests/[TestFileName].txt
ex: java demoA tests/puzzles03.txt

For DemoB (Sokoban) here is the format :

java demoB Tests/[TestFileName].txt
ex: java TP1B Tests/sokoban03.txt
